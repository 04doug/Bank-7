# Banco 7 - Plataforma Bancária Digital

![Banco 7](https://img.shields.io/badge/Banco-7-7b2cbf?style=for-the-badge&logo=bank&logoColor=white)
![Status](https://img.shields.io/badge/Status-Completo-success?style=for-the-badge)
![Tecnologias](https://img.shields.io/badge/Tecnologias-HTML5%2FCSS3%2FJavaScript-orange?style=for-the-badge)

Uma plataforma bancária digital moderna e responsiva que simula as principais funcionalidades de um banco digital real, desenvolvida com tecnologias web frontend.

## 🚀 Demonstração

[![Acessar Demo](https://img.shields.io/badge/ACESSAR%20DEMO-LIVE%20PREVIEW-blue?style=for-the-badge&logo=google-chrome)](https://github.com/04doug/Bank-7.git)

## ✨ Funcionalidades

### 🏠 Dashboard Principal
- **Saldo em tempo real** com toggle de visibilidade
- **Bitcoin tracker** com atualização automática de preços
- **Cofre de investimentos** com rendimento automático
- **Transações recentes** com histórico detalhado
- **Menu de ações rápidas** para funcionalidades principais

### 💰 Sistema de Investimentos
- **Múltiplas categorias:** Ações, Fundos, Criptomoedas, Imóveis
- **Portfólio pessoal** com acompanhamento de lucro/prejuízo
- **Detalhes completos** de cada investimento em modal
- **Simulação de mercado** com variações de preços realistas
- **Cofre automático** que rende periodicamente

### 🔄 Área PIX
- **Envio de PIX** com validação de saldo
- **Simulação de recebimentos** automáticos
- **Histórico de transações** detalhado
- **Interface intuitiva** para transferências

### 💳 Sistema de Cartões
- **Cartão de Crédito** Visa com design realista
- **Cartão de Débito** Mastercard
- **Informações detalhadas** de cada cartão
- **Interface visual** atrativa e profissional

### 👤 Perfil do Usuário
- **Edição de dados pessoais** em tempo real
- **Avatar personalizado** com iniciais
- **Salvamento automático** das alterações
- **Interface responsiva** para edição

### 🔔 Sistema de Notificações
- **Notificações em tempo real** para transações
- **Badge animado** para novos avisos
- **Painel lateral** com histórico completo
- **Atualizações automáticas** a cada 15 minutos

## 🔄 Funcionalidades em Tempo Real

### Simulações Ativas
1. **Preço do Bitcoin** - Atualiza a cada 30 segundos
2. **Recebimento PIX** - Novos valores a cada 15 minutos
3. **Rendimento do Cofre** - Crescimento automático a cada minuto
4. **Variação de Ações** - Preços atualizados periodicamente

### Estado da Aplicação
- Saldo bancário e transações
- Portfólio de investimentos
- Notificações e alertas
- Dados do usuário

## 🚀 Como Executar

1. **Clone o repositório:**
```bash
git clone https://github.com/04doug/Bank-7.git
```

2. **Acesse o diretório:**
```bash
cd banco-7
```

3. **Abra no navegador:**
```bash
# Método 1: Abrir diretamente o arquivo
open index.html

# Método 2: Usar um servidor local
python -m http.server 8000
# Acesse: http://localhost:8000
```
